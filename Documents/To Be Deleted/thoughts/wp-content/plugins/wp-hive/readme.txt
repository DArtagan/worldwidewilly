WP Hive is a WordPress Plugin that allows multiple WordPress sites to run on a single installation.

== Description ==

WP Hive is a WordPress Plugin that allows multiple WordPress sites to run on a single installation.

It is intended to be used in a multi-site, single administrator environment and is for the WordPress administrator who manages many sites and is looking for a way to synchronize the plugins, themes, and files for all their sites with ease.

The admin interface makes it incredibly easy to add new sites to the ‘Hive’.

Some features of WP Hive:

- Plugin architecture, simple to install
- Compatible with the latest releases of WordPress
- Compatible with almost all third party plugins
- Supports adding subdomain sites
- Supports wildcard domains and subdomains
- Supports adding subdirectory sites
- Server / OS agnostic

If you administer more than one WordPress site, WP Hive will make your life a lot easier!
